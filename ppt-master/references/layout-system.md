# 专业版式系统

## 版式核心目的

```
1. 传递信息 → 让内容更易读
2. 创造节奏 → 留白和密度的变化
3. 引导视线 → 视觉流动
4. 表达风格 → 传达主题调性
```

---

## 网格系统

### 常用网格

| 网格 | 适用场景 | 示例 |
|------|----------|------|
| 单栏 | 简单内容 | 标题+大图 |
| 双栏 | 图文左右排版 | 40:60或50:50 |
| 三栏 | 多列内容 | 三等分 |
| 栏栅 | 复杂混排 | 侧边栏+主内容+次内容 |
| 稻妻 | 超长标题 | 左右不对称 |

### 栏栅网格（复杂混排）

```
┌────┬────────────┐
│    │            │
│ 侧 │   主内容    │
│ 边 │            │
│    ├────┬───────┤
│ 栏 │col2│  col3 │
└────┴────┴───────┘

适用：后台界面、数据看板、复杂报告
```

### 稻妻网格（不对称）

```
┌──────────┬──────────┐
│          │          │
│   大列   │ 小列     │
│          │ 小列     │
│          │          │
├──────────┼──────────┤
│ 小列     │  大列    │
│ 小列     │          │
└──────────┴──────────┘

适用：打破常规、创意展示、视觉冲击
```

### PowerPoint参考线设置

```markdown
路径：视图 → 参考线 → 勾选

常用设置：
• 上下边距：1cm
• 左右边距：1.5cm
• 栏间距：0.5cm

技巧：添加额外的参考线来标记栏边界
```

## 封面页版式

### 1. 居中式（正式）

```
┌─────────────────────────┐
│                         │
│                         │
│        主  标  题        │
│        副  标  题        │
│        日期/姓名         │
│                         │
└─────────────────────────┘
```

**适用：** 学术演讲、正式汇报、官方发布

### 2. 左右分栏式（商务）

```
┌─────────────────────────┐
│                  │       │
│                  │ 主标题 │
│    装饰区域       │───────│
│    （图片/图形）  │ 副标题 │
│                  │ 机构名 │
└─────────────────────────┘
```

**适用：** 企业介绍、产品展示、项目汇报

### 3. 全图型（创意）

```
┌─────────────────────────┐
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│
│▓    主  标  题         ▓│
│▓                       ▓│
│▓    副  标  题         ▓│
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│
```

**适用：** 创意演讲、故事讲述、活动策划

### 4. 上下分割式（培训）

```
┌─────────────────────────┐
│                         │
│        标题区域          │
│       （30-40%）         │
├─────────────────────────┤
│                         │
│        装饰/图片         │
│       （60-70%）         │
│                         │
└─────────────────────────┘
```

**适用：** 培训课件、课程设计、在线教育

**要点：**
- 标题区占上部30-40%
- 可添加渐变或色块区分
- 下部用于图片或图形装饰

## 内容页版式

### 1. 标准标题栏

```
┌─────────────────────────┐
│ 页面标题                │
├─────────────────────────┤
│                         │
│        内容区域          │
│                         │
│                         │
└─────────────────────────┘
```

**适用：** 通用场景、最安全

### 2. 左右分栏

```
┌─────────────────────────┐
│ 页面标题                │
├────────────┬────────────┤
│            │            │
│   左侧     │    右侧    │
│  （图片）  │   （文字）  │
│            │            │
└────────────┴────────────┘
```

**适用：** 图文说明、对比展示

### 3. 上下内容

```
┌─────────────────────────┐
│        大图区域          │
│                         │
├─────────────────────────┤
│        文字说明          │
└─────────────────────────┘
```

**适用：** 大图配说明、多图并排

### 4. 三栏并列

```
┌───────┬───────┬───────┐
│       │       │       │
│ 要点1 │ 要点2 │ 要点3 │
│       │       │       │
└───────┴───────┴───────┘
```

**适用：** 特点展示、优势对比

### 5. 卡片式

```
┌─────┐  ┌─────┐  ┌─────┐
│     │  │     │  │     │
│ 卡片 │  │ 卡片 │  │ 卡片 │
│     │  │     │  │     │
└─────┘  └─────┘  └─────┘
```

**适用：** 多项目展示、服务介绍

### 6. 时间轴

```
●──────────●──────────●──────────●
│          │          │          │
│  阶段1   │  阶段2   │  阶段3   │
```

**适用：** 流程展示、发展历程

## 章节过渡页

### 版式要点

```
大号章节号 + 大标题 + 可选副标题

特点：
• 章节号最大（72-120pt）
• 标题次大（36-48pt）
• 内容最少（只有章节号+标题）
• 大量留白
```

## 结束页版式

| 类型 | 适用场景 | 特点 |
|------|----------|------|
| 感谢式 | 正式演讲 | 简洁、感谢语+联系方式 |
| Q&A式 | 讨论环节 | 问答环节、欢迎提问 |
| 行动号召 | 销售提案 | CTA按钮、立即行动 |
| 升华式 | 演讲结尾 | 金句、情感共鸣 |

## 版式选择指南

| 场景 | 推荐版式 |
|------|----------|
| 企业汇报 | 标准标题栏、居中式 |
| 产品发布 | 全图型、左右分栏 |
| 培训课件 | 上下分割、卡片式 |
| 学术演讲 | 居中式、标准标题栏 |
| 销售提案 | 左右分栏、行动号召 |
| 项目汇报 | 标准标题栏、多栏并列 |
| 个人展示 | 全图型、居中式 |

## 版式常见错误

| ❌ 错误 | ✅ 正确 |
|---------|---------|
| 内容拥挤 | 内容占60-70%，留白30-40% |
| 对齐混乱 | 使用参考线 |
| 层次不清 | 标题与正文2倍差距 |
| 视觉疲劳 | 交替使用不同版式 |
