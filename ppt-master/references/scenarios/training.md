# 培训课件场景指南

> 适用：教学、培训、workshop、知识传授

---

## 场景特征

| 特征 | 说明 |
|------|------|
| **PPT等级** | 二等 - 逻辑层 |
| **信息密度** | 中（适当填充） |
| **观众特点** | 需要学习、会拿PPT当资料 |
| **核心目标** | 传授知识、让人记住、能用 |

---

## 核心设计原则

### ✅ 应该做的

| 原则 | 说明 | 示例 |
|------|------|----------|
| 结构清晰 | 模块化知识体系 | 目录+章节过渡 |
| 案例丰富 | 理论联系实际 | 每个概念配案例 |
| 循序渐进 | 从易到难 | 渐进式学习路径 |
| 互动设计 | 让学生参与 | 思考题、练习 |
| 可当资料 | 学生会保存PPT | 适当详细 |

### ❌ 不应该做的

| 错误 | 说明 |
|------|------|
| 太理论 | 听不懂、记不住 |
| 太枯燥 | 走神 |
| 信息过载 | 塞太满 |
| 没有互动 | 学生只是听 |

---

## 与乔布斯式的区别

| 培训课件 | 乔布斯式发布 |
|----------|--------------|
| 传授知识 | 打动人心 |
| 模块化结构 | 故事线 |
| 可以详细 | 必须极简 |
| 需要案例 | 需要震撼 |
| 学生保存PPT | PPT过耳即忘 |

## 版式风格

**推荐风格**：Soft / Rounded

| 元素 | 数值 |
|------|------|
| 圆角 | 0.08" ~ 0.15" |
| 页面边距 | 0.4" ~ 0.5" |
| 元素间距 | 0.15" ~ 0.3" |
| 留白比例 | 30-40% |

---

## 黄金结构

```
第1页：封面（主题 + 学习目标）
第2页：目录（今天讲什么）
第3页：模块1导入（为什么学这个）
内容页：模块1 - 概念讲解
内容页：模块1 - 案例分析
内容页：模块1 - 练习/思考
第X页：模块2 - 同上结构
第X页：模块3 - 同上结构
互动页：综合练习
总结页：核心要点回顾（3-5点）
结束页：下一步行动 + 资源
```

---

## 内容密度控制

| 页面类型 | 要点数量 | 说明 |
|----------|----------|------|
| 概念页 | 1-2个 | 一个概念讲透 |
| 案例页 | 2-3个 | 案例的关键点 |
| 练习页 | 1-2个 | 一个练习讲透 |
| 总结页 | 3-5个 | 核心要点 |

---

## 互动设计技巧

| 互动类型 | 时机 | 作用 |
|----------|------|------|
| 思考题 | 每个模块前 | 引发思考 |
| 练习 | 每个模块后 | 巩固知识 |
| 提问 | 讲解中 | 保持注意力 |
| 讨论 | 复杂概念前 | 促进理解 |

---

## 检查清单

```
□ 开头是否明确学习目标？
□ 结构是否模块化（学生能找到）？
□ 每个概念是否配案例？
□ 是否有互动环节（思考题/练习）？
□ 难度是否循序渐进？
□ 内容是否适合当资料（不过于简略）？
□ 配色是否舒适（不是太花哨）？
□ 字体是否清晰可读？
□ 页数是否适当（不是越长越好）？
□ 结尾是否有行动建议？
```
