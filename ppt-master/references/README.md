# references 目录索引

> 这是ppt-master技能的文档索引入口。告诉你每个文件在哪里、什么时候该读。

---

## 目录结构

```
references/
├── README.md              ← 你在这里
├── cases/                 ← 案例库（好案例/坏案例）
├── guide/                 ← 核心方法论
├── scenarios/             ← 场景指南
└── [其他文件]             ← 单文件文档
```
