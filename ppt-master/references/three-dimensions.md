# PPT三维体系：难忘感·设计感·仪式感

> 本章源自《PPT之光》（冯注龙），提供评估PPT整体质量的完整框架。
> 与 philosophy.md（核心理念）、techniques.md（实操技巧）形成互补。

---

## 1. 三维体系概述

PPT制作需要从三个维度综合考量：

| 维度 | 核心 | 占比建议 | 关键问题 |
|------|------|----------|----------|
| **难忘感** | 让人记住你 | 40% | 观众离开后能记住什么？ |
| **设计感** | 让PPT好看 | 40% | PPT是否专业好看？ |
| **仪式感** | 让观众投入 | 20% | 演讲是否有感染力？ |

### 1.1 三维体系快速自检

完成PPT后，用以下问题快速检查：

```
□ 难忘感：有没有一个记忆点（画面/数据/金句）？
□ 设计感：配色和排版是否全程一致？
□ 仪式感：开头是否吸引？结尾是否有升华？
```

## 2. 难忘感 - 让人记住你

> **核心问题**：观众离开后能记住什么？

### 2.1 记忆点类型

| 类型 | 说明 | 例子 | 适用场景 |
|------|------|------|----------|
| **金句** | 一句话让人记住核心观点 | "DT时代，数据就是石油" | 核心观点输出 |
| **画面** | 视觉冲击让人难忘 | 数据可视化"数字树" | 视觉冲击 |
| **故事** | 情感共鸣 | 创始人逆袭故事 | 品牌/产品介绍 |
| **对比** | 强烈反差 | "Before vs After" | 解决方案展示 |
| **数据** | 具体数字 | "增长320%" | 成果展示 |

### 2.2 开场设计（四招）

开场决定观众的第一印象，决定是否继续听下去。

| 开场方式 | 适用场景 | 示例 | 效果 |
|----------|----------|------|------|
| **自我介绍** | 首次演讲 | "我是XXX，专注XXX领域10年" | 建立信任 |
| **故事导入** | 情感共鸣 | "分享一个真实案例..." | 情感连接 |
| **问题导入** | 引发思考 | "你们有没有遇到过..." | 激发好奇 |
| **数据导入** | 增强说服 | "过去一年，我们发现..." | 数据震撼 |

#### 开场5秒法则

| 场景 | ❌ 错误开场 | ✅ 正确开场 |
|------|------------|-------------|
| 商业汇报 | "今天我汇报..." | "Q3收入增长200%，核心原因是..." |
| 产品发布 | "今天发布..." | "想象一下，以后你出门只需要..." |
| 培训 | "今天讲..." | "学完这节课，你能..." |
| 学术 | "我的论文是..." | "这个问题，困扰了学界10年..." |

### 2.3 五种强效标题类型

好的标题让观众一眼就知道能得到什么。

| 类型 | 特征 | 示例 |
|------|------|------|
| **干货型** | 直接点明价值 | 《2024年Q1业绩增长的5个关键策略》 |
| **悬念型** | 引发好奇 | 《为什么90%的营销预算都浪费了？》 |
| **数字型** | 具体可量化 | 《3步教你写出10万+阅读量的文章》 |
| **对比型** | 制造冲突 | 《传统营销 vs 数字营销：胜负已分》 |
| **打破常规型** | 颠覆认知 | 《努力工作是最愚蠢的职场策略》 |

### 2.4 结尾设计（三招）

结尾是最后的记忆点，决定了演讲的余韵。

| 结尾类型 | 效果 | 示例 |
|----------|------|------|
| **总结升华** | 强化记忆 | "让我们总结一下今天的核心要点" |
| **行动号召** | 促进转化 | "现在就行动，立即申请..." |
| **金句收尾** | 余音绕梁 | "记住：..." |
| **感谢收尾** | 礼貌结束 | "感谢大家的聆听" |
| **Q&A互动** | 解答疑惑 | "有什么问题，欢迎提问" |

#### 结尾Call to Action

| 场景 | 结尾应该 |
|------|----------|
| 商业汇报 | "请领导批准..." |
| 商业提案 | "期待合作，具体方案..." |
| 产品发布 | "立即购买/预约..." |
| 培训 | "回去后请完成..." |
| 学术答辩 | "恳请各位老师批评指正" |

## 4. 仪式感 - 让观众投入

> **核心问题**：演讲是否有感染力？
>
> **详细说明**：演讲准备的具体内容请参考 [speech-preparation.md](speech-preparation.md)

### 4.1 什么是仪式感

让观众感受到"这是一个正式的、有分量的演示"。

### 4.2 如何塑造仪式感

| 方法 | 操作 |
|------|------|
| **专业的开场** | 准时开始，正式自我介绍 |
| **统一的风格** | 配色、字体全程一致 |
| **适当的停顿** | 重要观点前停顿 |
| **眼神交流** | 与观众建立连接 |
| **肢体语言** | 适度走动，手势配合 |

### 4.3 通俗表达

| 技巧 | 应用 |
|------|------|
| **善于类比** | 复杂概念用通俗类比解释 |
| **大白话** | 少用专业术语，让观众易懂 |
| **讲故事** | 用故事阐述观点，更有说服力 |
| **幽默** | 适当幽默，让气氛轻松 |

## 5. 三维体系自检清单

完成PPT后，逐项检查：

### 难忘感检查
- [ ] 有没有一个记忆点（画面/数据/金句）？
- [ ] 记忆点是否容易复述？
- [ ] 开头是否在5秒内吸引注意力？
- [ ] 结尾是否有升华或明确行动号召？

### 设计感检查
- [ ] 配色是否统一（不超过3种颜色）？
- [ ] 字体是否规范（标题vs正文对比清晰）？
- [ ] 排版是否有序（对齐、亲密）？
- [ ] 图片是否清晰、相关？

### 仪式感检查
- [ ] 开场是否专业？
- [ ] 演讲是否有感染力（停顿、重音、互动）？
- [ ] 是否有过渡语设计？
- [ ] 整体是否让观众感到"这是一个正式的演示"？
