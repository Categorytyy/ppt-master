# 专业配色系统

## 配色基础

### 三要素

| 要素 | 说明 | 在PPT中的应用 |
|------|------|----------------|
| 色相 | 颜色的本质（红、绿、蓝等） | 选择主色 |
| 明度 | 颜色的明暗程度 | 调整深浅搭配 |
| 纯度 | 颜色的鲜艳程度 | 强调vs辅助 |

### 色调分类

| 色调 | 颜色 | 情绪 | 适用场景 |
|------|------|------|----------|
| 暖色调 | 红、橙、黄 | 活力、热情、紧迫 | 促销、儿童、活动 |
| 冷色调 | 蓝、绿、紫 | 专业、平静、科技 | 商务、科技、医疗 |
| 中性色 | 黑、白、灰 | 百搭、过渡 | 背景、文本 |

## 配色比例法则

### 7:2:1法则

```
主色：70% - 视觉主体、背景
辅助色：20% - 次要元素
强调色：10% - 重点强调
```

### 实际应用（Theme映射）

```javascript
// 示例：商务蓝配色
const theme = {
  primary: "1A237E",    // 深蓝 - 主色70%
  secondary: "3949AB",  // 中蓝 - 辅助色20%
  accent: "F57C00",     // 橙色 - 强调色10%
  light: "E8EAF6",      // 浅蓝 - 浅色装饰
  bg: "FFFFFF"          // 白色 - 背景
};
```

## 配色方案类型

### 1. 单色系配色（最安全）

```markdown
主色：蓝色 #2196F3
辅助色：深蓝 #1565C0 + 浅蓝 #64B5F6
文本色：深灰 #333333
背景色：白色 #FFFFFF

适用：企业汇报、商务演示、学术演讲
```

### 2. 互补色配色（最强烈）

```markdown
主色：蓝色 #2196F3
对比色：橙色 #FF9800
文本色：深灰 #333333
背景色：白色 #FFFFFF

适用：需要强调、突出对比的场景
慎用：整体配色不要五五开，7:3或8:2比例
```

### 3. 类比色配色（最和谐）

```markdown
主色：蓝色 #2196F3
辅助色：青绿 #00BCD4 + 紫蓝 #3F51B5
文本色：深灰 #333333
背景色：白色 #FFFFFF

适用：科技、现代、专业主题
```

### 4. 中性色+点缀色（最优雅）

```markdown
背景色：深灰 #2D2D2D 或 浅灰 #F5F5F5
点缀色：金色 #FFD700 或 珊瑚色 #FF6F61
文本色：白色 #FFFFFF 或 深灰 #333333

适用：高端演示、艺术主题、摄影展示
```

## 配色实战技巧

### 从品牌提取配色

```markdown
1. 找到品牌主色
2. 确定主色、辅助色、文本色比例
3. 使用拾色器工具提取HEX值
4. 按7:2:1原则分配比例
```

### 配色避坑

```markdown
❌ 忌讳：
• 颜色超过5种
• 高饱和度颜色占比过大
• 背景和文字对比度不足
• 颜色平均分配没有重点

✅ 正确：
• 控制在3种颜色+黑白灰
• 主色占比60-70%
• 对比度≥4.5:1
• 7:2:1或8:1:1比例
```

### 对比度检查工具

```markdown
• WebAIM Contrast Checker（检查对比度）
• Coolors.co（在线配色生成）
• Adobe Color（配色方案提取）
```

### 字号规范

| 元素 | 字号 | 说明 |
|------|------|------|
| 封面标题 | 48-72pt | 最大 |
| 章节标题 | 28-36pt | 页面标题 |
| 副标题 | 18-24pt | 辅助说明 |
| 正文 | 14-18pt | 主体内容 |
| 注释 | 10-12pt | 最小 |

### 字体对比原则

```markdown
标题vs正文：
• 字号差至少6pt以上
• 粗细要有区分（Bold vs Regular）
• 字体可以不同（但保持协调）

❌ 错误对比：
标题20pt Bold vs 正文18pt Regular → 差距太小
标题24pt Regular vs 正文24pt Regular → 无粗细对比

✅ 正确对比：
标题36pt Bold vs 正文16pt Regular
标题32pt Bold vs 正文14pt Regular
```

### 字体行距

```markdown
PPT正文的行距：1.2-1.5倍
不要使用单倍行距（太拥挤）
不要使用2倍以上（太分散）
```

## 配色严格规则（MANDATORY）

**⚠️ 只使用提供的调色板，禁止创建或修改颜色**

```markdown
✅ 正确做法：
• 所有颜色必须来自用户提供的调色板
• 不使用调色板外的颜色
• 不修改调色板颜色（亮度、饱和度、混合）

❌ 错误做法：
• "大概调深一点" → 禁止
• "这个蓝不够好看，换一个" → 禁止
• "混一点灰色进去" → 禁止
```
