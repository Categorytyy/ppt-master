# PPT完整质量检查清单

> 本清单整合了ppt-master和ppt设计技能的质量检查标准。
> 完成PPT制作后，逐项检查确保质量。

---

## 一、内容检查

| 检查项 | 标准 | 通过 |
|--------|------|------|
| 每页核心信息 | 每页只有一个核心信息 | □ |
| 文字精炼 | 文字精炼，无废话 | □ |
| 逻辑清晰 | 逻辑清晰，层次分明 | □ |
| 开头吸引力 | 开头有吸引力，能在5秒内抓住注意力 | □ |
| 结尾记忆点 | 结尾有记忆点（总结/行动号召/金句） | □ |
| 标题质量 | 标题即结论，不是模糊空泛 | □ |
| 无错别字 | 无错别字、语法错误 | □ |

### 标题自检

- [ ] 标题是否让人一眼知道能得到什么？
- [ ] 是否使用了五种强效标题之一（干货型/悬念型/数字型/对比型/打破常规型）？
- [ ] 标题与内容是否匹配？

## 二、设计检查

### 2.1 配色检查

| 检查项 | 标准 | 通过 |
|--------|------|------|
| 颜色数量 | 不超过3种颜色（+黑白灰） | □ |
| 主色占比 | 主色占比60-70% | □ |

## 八、检查流程建议

### 制作过程中
1. 每完成一页，检查一页
2. 重点检查：配色、字体、对齐

### 完成后整体检查
1. 内容检查（是否每页只有一个核心）
2. 设计检查（配色、字体、排版）
3. 技术检查（代码、文件）
4. 三维体系最终确认

### 演讲前检查
1. 文件能正常打开
2. 投影显示正常
3. 动画效果正常（手动添加的动画）

## 九、常见问题修复

| 问题 | 修复方法 |
|------|----------|
| 内容太多 | 每页只留一个核心信息，其他放到备注 |
| 标题模糊 | 用五种强效标题类型重写 |
| 配色混乱 | 确定主色（60-70%）+辅助色（20-30%）+强调色（5-10%） |
| 字体不统一 | 选择2种字体组合，全程使用 |
| 排版杂乱 | 使用参考线对齐，相近内容放一起 |
