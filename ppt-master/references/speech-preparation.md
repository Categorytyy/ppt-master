# 演讲准备完整指南

> 本章帮助做好演讲前的充分准备，确保演讲质量。
> 与 [three-dimensions.md](three-dimensions.md) 的"仪式感"部分配合使用。

---

## 1. 演讲准备清单

### 1.1 内容准备

- [ ] 熟悉内容（至少练习3遍）
- [ ] 准备演讲稿/提示词
- [ ] 设计过渡语
- [ ] 设计开场白（5秒内抓住注意力）
- [ ] 设计结尾（总结/行动号召/金句）
- [ ] 准备可能被问的问题及答案

### 1.2 物质准备

- [ ] 确认电脑正常运行
- [ ] 确认投影显示正常
- [ ] 翻页笔电池充足
- [ ] 音响（如需要）正常
- [ ] 激光笔（如需要）正常
- [ ] 有备份文件（U盘+云端）
- [ ] 准备纸质笔记/卡片（备用）

### 1.3 心态准备

- [ ] 调整心态，与紧张和平共处
- [ ] 提前到场熟悉环境
- [ ] 预演一遍

## 2. 应对紧张

### 2.1 紧张是正常的

> 紧张是身体的自然反应，适度的紧张能让你表现更好。

### 2.2 应对方法

| 方法 | 说明 | 何时使用 |
|------|------|----------|
| 深呼吸 | 上台前做3次深呼吸 | 候场时 |
| 目光接触 | 与友善的观众眼神交流 | 演讲中 |
| 放慢语速 | 紧张时容易越说越快，有意识放慢 | 演讲中 |
| 适当走动 | 移动可以释放紧张情绪 | 演讲中 |
| 准备开场 | 好的开场能建立信心 | 演讲前 |
| 与紧张共处 | 接受紧张，带着紧张继续 | 演讲中 |

### 2.3 紧张时的具体做法

```
1. 深呼吸3次（吸气4秒，屏住2秒，呼气6秒）
2. 微笑（即使不笑，嘴角上扬能缓解紧张）
3. 看向友善的观众（通常在前排或两侧）
4. 从最熟悉的内容开始讲
5. 手里握着翻页笔，增加安全感
```

## 3. 演讲技巧

### 3.1 核心技巧

| 技巧 | 应用 | 效果 |
|------|------|------|
| **停顿** | 重要观点前停顿，制造悬念 | 强调重点 |
| **重音** | 关键词加重音，强调重点 | 突出关键 |
| **重复** | 重要内容重复，加深印象 | 强化记忆 |
| **互动** | 提问、调查、现场互动 | 提升参与感 |
| **故事** | 用故事阐述观点 | 增强说服力 |
| **类比** | 复杂概念用通俗类比解释 | 易于理解 |

### 3.2 停顿的运用

| 场景 | 停顿时长 | 效果 |
|------|----------|------|
| 重要观点前 | 2-3秒 | 制造悬念 |
| 转折时 | 1-2秒 | 提示逻辑变化 |
| 幽默后 | 1秒 | 给观众反应时间 |
| 提问后 | 3-5秒 | 给观众思考时间 |

### 3.3 互动的设计

| 互动类型 | 示例 | 适用场景 |
|----------|------|----------|
| 提问 | "你们有没有遇到过..." | 引发思考 |
| 调查 | "觉得OK的举手" | 了解认同度 |
| 讨论 | "两人一组讨论1分钟" | 提升参与 |
| 案例 | "看这个案例..." | 增强说服 |

## 4. 过渡语设计

### 4.1 过渡语的作用

过渡语连接前后内容，让演讲流畅自然。

### 4.2 过渡语示例

| 从 | 到 | 过渡语示例 |
|----|----|------------|
| 开场 | 第一章节 | "首先，让我们来看看..." |
| 章节1 | 章节2 | "了解了XX之后，我们来看看..." |
| 章节2 | 章节3 | "那么，如何实现呢？" |
| 内容 | 总结 | "综上所述，我们可以得出..." |
| 总结 | 结束 | "最后，我想用一句话总结..." |
| 结束 | Q&A | "以上就是我的分享，欢迎提问..." |

### 4.3 过渡语设计检查

```
□ 是否每个章节之间有过渡语？
□ 过渡语是否自然流畅？
□ 过渡语是否提示了逻辑关系（因果、递进、对比）？
```

## 5. FAQ准备

### 5.1 常见问题准备

| 问题类型 | 示例问题 | 回答要点 |
|----------|----------|----------|
| 数据来源 | "这个数据从哪来的？" | 说明权威来源 |
| 竞争对手 | "和竞品比有什么优势？" | 突出差异化 |
| 实施细节 | "具体怎么落地？" | 简述步骤 |
| 预算/时间 | "需要多久/多少钱？" | 给出范围 |

### 5.2 FAQ准备格式

```
问题1：[关于数据的来源？]
简短回答（30秒内）："数据来自XXX机构的2024年报告，样本量是..."
备用数据："如果没有这个数据，我们还有..."

问题2：[关于竞争对手的对比？]
简短回答（30秒内）："我们和XX的核心差异是..."
优势点："我们的优势在于..."

问题3：[关于实施计划的细节？]
简短回答（30秒内）："大致分三个阶段..."
时间线："预计X个月完成"
```

## 6. 设备检查清单

### 6.1 演讲前检查

| 检查项 | 具体内容 | 状态 |
|--------|----------|------|
| 电脑 | 是否正常开机？电量是否充足？ | □ |
| 投影 | 显示是否正常？分辨率是否正确？ | □ |
| 翻页笔 | 电池是否充足？是否可用？ | □ |
| 音响 | 声音是否正常？（如有音频） | □ |
| 网络 | 是否需要网络？（如在线演示） | □ |
| 文件 | PPT文件是否在桌面上？能否打开？ | □ |
| 备份 | 是否有U盘备份？云端是否上传？ | □ |

### 6.2 设备应急方案

| 问题 | 应急方案 |
|------|----------|
| 电脑死机 | 立即重启，使用备份文件 |
| 投影不显示 | 检查连接线，切换投影模式（Win+P） |
| 文件损坏 | 使用U盘/云端备份 |
| 声音没声音 | 检查音量、音频文件是否嵌入 |

## 7. 开场和结尾设计

### 7.1 开场设计（5秒法则）

> 开头5秒决定观众是否继续听。

| 场景 | ❌ 错误开场 | ✅ 正确开场 |
|------|------------|-------------|
| 商业汇报 | "今天我汇报..." | "Q3收入增长200%，核心原因是..." |
| 产品发布 | "今天发布..." | "想象一下，以后你出门只需要..." |
| 培训 | "今天讲..." | "学完这节课，你能..." |
| 学术 | "我的论文是..." | "这个问题，困扰了学界10年..." |

### 7.2 结尾设计

| 结尾类型 | 效果 | 示例 |
|----------|------|------|
| **总结升华** | 强化记忆 | "让我们总结一下今天的核心要点" |
| **行动号召** | 促进转化 | "现在就行动，立即申请..." |
| **金句收尾** | 余音绕梁 | "记住：..." |
| **感谢收尾** | 礼貌结束 | "感谢大家的聆听" |
| **Q&A互动** | 解答疑惑 | "有什么问题，欢迎提问" |

## 8. 演讲自检

演讲前，用以下问题自检：

```
□ 内容是否熟悉？（能脱离PPT讲）
□ 开场是否准备好？（5秒内吸引注意力）
□ 过渡语是否设计好？
□ 时间控制是否合理？
□ 可能的问题是否准备好？
□ 设备是否检查过？
□ 心态是否调整好？
```
